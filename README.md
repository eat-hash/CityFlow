# CityFlow 多路口交通信号智能优化系统
《Web应用程序开发》课程设计项目

## 项目介绍
实现多路口统一管理、模拟交通数据批量导入、一键生成独立配时方案、优化效果可视化对比的完整闭环。每个路口独立计算，不涉及跨路口协同控制。

## 技术栈
- 前端：HTML5 + CSS3 + JavaScript + ECharts 5.0
- 后端：Java 8 + Servlet 3.1 + JSP 2.3
- 数据库：MySQL 5.7 + JDBC
- 部署：Tomcat 9.0

## 运行说明
1.  安装 JDK 8、Tomcat 9.0、MySQL 5.7
2.  执行 `sql/create_tables.sql` 创建数据库表
3.  执行 `sql/init_data.sql` 初始化预设路口数据
4.  将项目导入IDEA，配置Tomcat服务器
5.  启动Tomcat，访问 http://localhost:8080/CityFlow
