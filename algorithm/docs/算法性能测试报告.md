# 算法性能测试报告
## 1.测试环境
JDK8/11，Windows本地环境，CPU常规办公处理器
测试样本：10个独立路口，四相位固定车流

## 2.测试用例
用例1：默认参数 minGreen=15s，maxCycle=120s
用例2：自定义参数 minGreen=20s，maxCycle=150s

## 3.测试结果
|路口数量|配置参数|耗时(ms)|是否达标(＜1000ms)|
| ---- | ---- | ---- | ---- |
|10|默认15/120|xxx|是|
|10|自定义20/150|xxx|是|

## 4.优化说明
1.TrafficSignalOptimizer内部全局复用Random对象，取消循环频繁new Random，减少对象创建开销
2.BatchOptimizer预分配List容量，减少集合扩容开销
3.批量构造Input时复用Flow Map，减少重复Map初始化

## 5.对接说明
后端可调用AlgorithmService两个入口：
1.单个：optimizeSingle(id,flow,minG,maxC)
2.批量：optimizeBatch(idList,flow,minG,maxC)
